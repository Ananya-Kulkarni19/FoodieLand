## Project Setup

### Prerequisites

- Node.js
- npm or yarn
- Git

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/baqx/foodieland
   cd foodieland

   ```

2. Install all packages

   ```bash
   npm install
   ```

3. Start the server
   ```bash
   npm start
   ```
